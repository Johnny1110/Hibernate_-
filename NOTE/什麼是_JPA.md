# 什麼是 JPA ?

<br>


## Java Persistence API ( JPA )

* JPA 屬於 Java EE 標準。是 Java 針對 ORM ( 物件關聯對映 ) 所制定的一套介面規範，簡單說就是，Sun 公司說現在市面上一堆 ORM，那就來制定一套規範統一一下所有框架吧。

* JPA 的制定式 Hibernate 的開發者們參與訂製的，所以 JPA 可以說是 Hibernate 目前最為推崇大家使用的規範。
